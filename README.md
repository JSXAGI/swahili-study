# スワヒリ語学習アプリ：全ての始まり・AIたちとの共闘

> **ここにあるのは、私がAndroid Studioでスマホアプリ（APK）を完成させるための『心臓部』となったWebアプリのコードです。制作の舞台裏は以下の通りです。**

### 【開発の経緯】
全ては「Canvaのコードをキオスクで動かしたい」という夢から始まった。Claude 3.5 Sonnetに基盤を、ChatGPTにローカルサーバーとPythonを学び、5時間に及ぶ対話の末にキオスクを立ち上げた。

しかし、オフラインでのレイアウト崩れという壁に直面。ChatGPTの利用制限中、助けを求めたのがGeminiだった。彼と2時間の激闘の末、崩れたレイアウトを修正し、ついに「英語・スワヒリ語学習キオスク」が完成した。

ネット環境のないタンザニアの子どもたちに、スマホでこの学びを届けたい。その一心で、Geminiを唯一の相棒に「Website 2 APK Builder」や「Android Studio」という未知の世界へ飛び込み、最初のアプリアイコンまで作り上げた記念すべき第一作。
